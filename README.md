# hoas_kitchen_helper
Die Küchenhelfer-Integration für Home Assistant verbindet Essensplanung und Einkaufsliste in einem übersichtlichen Dashboard. Per HACS installiert, bietet sie smarte Karten für Rezeptideen, KI-gestützte Vorschläge bei unbekannten Gerichten und fügt automatisch alle benötigten Zutaten der Home Assistant Einkaufsliste hinzu.
